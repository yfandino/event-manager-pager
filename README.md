# Event manager (Pager)

## Requisites

Node.js 14.0.0 or later - [Download & Install Node.js](https://nodejs.org/en/download/) and the npm package manager.


## Installation

Use the package manager [npm](https://www.npmjs.com/) to install the project.

```bash
npm install
```

## Test

```bash
npm test
```
